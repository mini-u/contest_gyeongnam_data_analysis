## 2025.05.22 회의
### 할 일
1. 대시보드 레이아웃 짜기
2. 데이터 합치기

   2. 접근성: 버스정류장, 자전거정류장, 지하철역, 주차장
   3. 인구: 거주인구, 생활인구(수요 관련 지표)
